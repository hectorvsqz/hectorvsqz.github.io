So, if you're looking to create your own page in Github.com, follow these steps:

  1. Create a Github account, here you will have the chance to create a user name
  2. Next, create a repository, give it a name and append .github.io, so for example repo.github.io, or username.github.io
  3. In the repository, in the code tab, you can create, or drag and drop your website files - Github will search for files with the name index.html, index.md, or README.md, by default as entry files for your website
  4. You then have to commit the files to your repository
    • You can add images, text, videos, links, etc. as you wish
  5. Navigate to the website to see your results

Truthfully, there is more to this, but hope this gives you a good idea. Thanks for reading!
